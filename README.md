## カーレントディレクトリでファイルアプリを起動するコマンドの登録

1.設定ファイルを開く
```bash
open .bashrc
```

2.ファイルの末尾に以下の関数を追記して上書き保存する
```bash
file() {
    # 引数があればそのパスを、なければ現在のディレクトリ(.)を使用
    local target="${1:-.}"
    # 背景で実行し、標準出力・エラーを破棄、ターミナルから完全に分離
    (nautilus "$target" &> /dev/null &)
}
```
3.設定変更を反映させる
```bash
source ~/.bashrc
```

# Done!
